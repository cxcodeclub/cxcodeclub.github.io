Charing Cross Library

What we do

We're a free computer coding club for children aged 9-11 at Charing Cross Library in the heart of London's West-End. These are the programming languages we teach:

  1. Scratch
  2. Website markup (HTML & CSS)
  3. Python
  4. JavaScript

Here are more advanced projects for you to try:

  • Websites (HTML, CSS & JavaScript)
  • Advanced Websites (HTML, CSS, JavaScript & modern libraries)
  • Ruby

What's next?

Once you're comfortable with Python and JavaScript try practising your skills:

  • Wizzie Wizzie Computer Coding Club (for 8-14 year olds)
  • CoderDojo (for 7-17 year olds)
  • Young Rewired State (organises the free, week-long national Festival of Code hackathon the first week of the summer holidays)
  • Major League Hacking (free weekend hackathons around the UK through the autumn)
